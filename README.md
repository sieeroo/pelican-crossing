# Pelican Crossing System
Program ini adalah sistem lampu lalu lintas penyeberangan untuk pejalan kaki yang disimulasikan dengan arduino. 

## Teknologi yang Digunakan
* Bahasa Pemrograman : C++
* Website Simulasi : Tinkercad
* Konsep Pemrograman : 

## Bahan Untuk Membuat
* Arduino Uno x1
* LCD I2C x1
* Breadboard x2
* Resistor 
* Kabel Jumper
